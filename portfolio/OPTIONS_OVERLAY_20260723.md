## 0. 假說（主假說之期權變種）

主假說（升級的樹跑贏降級的樹）若成立，配對期權組合可將相對 alpha
乘以市場波動放大：

```
升級組 → 買 call（或 call debit spread）
降級組 → 買 put（或 put debit spread）
配對規則 → 兩邊 delta 平衡：Σ|call delta| ≈ Σ|put delta|
```

組合淨 delta ≈ 0——市場方向由兩邊凸性互相掩護（升市 call 之無限上行
對 put 之有限損失；跌市反之），不靠預測大盤。**弱點在橫行市**：兩邊
theta 齊燒，僅餘相對 alpha——故設入場閘（§3）。

## 1. 資金結構（2026-07-23 同日修訂：三倉架構，本 sleeve 33%）

- 帳戶分三個 sleeve（詳見 `STRATEGY_SLEEVES_20260723.md`）：
  **A Kelly 34%｜B Alpha 長短倉 33%｜C 期權配對（本檔）33%**。
- Sleeve C **每週權利金風險上限 ≤ 3%（以 sleeve 計）**；單一訊號
  ≤ 0.5%（sleeve 計）。33% 是會計分隔，非風險倍增——sleeve 大部分
  時間持現金。
- Sleeve C 設獨立 journal（`_paper/options/`）與獨立 base-100 指數；
  各實驗對答案互不污染。

## 2. 結構選單（按訊號分層）

| 層 | 升級側 | 降級側 |
|---|---|---|
| \|訊號\| ≥ 2.5 | 淨 call（ATM，2–4 週期） | 淨 put（ATM，2–4 週期） |
| 1.6 ≤ \|訊號\| < 2.5 | call debit spread（ATM／+10–15%） | put debit spread（ATM／−10–15%） |
| \|訊號\| < 1.6 | 不落注 | 不落注 |
| 📅 財報後事件 | 優先（IV crush 後入場、即時公開日翌日開盤） | 同左 |

- **加權規則（訊號分為權）**：兩邊內部之權利金與 delta 皆按 |訊號分|
  比例分配（訊號重者注碼大）；兩邊總 delta 校平。落賬同時記等權與
  訊號加權兩套歸因，與計分板之雙讀數（等權／加權策略回報）對齊——
  期權 sleeve 實際上執行的是**加權版**策略。
- 收租衛星（sell call spread 於預期橫行跑輸之降級股）：僅限
  ≤ 0.5%／週之小注，short leg 到期前有財報者不開。

## 3. 入場閘（每週開倉前計一次，落賬）

```
預期散度 ＝ 歷史組差均值（正式計分板累積值；樣本不足時用先導值並註明）
總成本   ＝ 兩邊權利金合計（以 implied move 口徑）
開倉條件 ＝ 預期散度 > 總成本 × 1.5
```

不達標 ⇒ 該週 Sleeve B 空倉（此決定同樣落賬——「不開倉」也是紀錄）。

## 4. 流動性閘（逐檔）

期權 OI ≥ 500 且 bid-ask ≤ 中間價 10% 方可入場；不合格者如實跳過並
落賬（預期 POET／CBRS 類小型股多數過不了此關——如實記錄即可）。

## 5. 離場（三則先到先行）

1. 該股判定 state 翻轉（樹的 thesis 已變）⇒ 即平；
2. 到期；
3. long 結構獲利 ≥ 3 倍 ⇒ 平一半。

到期選擇隨 H-horizon 曲線成熟而修訂（曲線見頂位＝最適到期）。
